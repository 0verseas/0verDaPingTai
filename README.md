# 0verDaPingTai

## usage
```
git clone https://github.com/0verseas/0verDaPingTai.git
cd 0verDaPingTai
git submodule init
git submodule update
npm install
cp src/env.js.example src/env.js
```
edit the config file in ``src/env.js``

## run
```
npm run serve
```

## compile
```
npm run build
```

## update submodule
```
git submodule update --recursive --remote
```
